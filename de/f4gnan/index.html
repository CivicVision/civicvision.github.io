Example Notebook Structure

  • Title
  • Short explanation why you‘re doing it.
  • Result (Visualization, Conclusion)
  • Steps to create with explanations in between (make good use of the markdown you‘re able to write)
  • Data Including Data Sources Explanations
  • Libraries
  • Date of the creation/update
